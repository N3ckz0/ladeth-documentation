Herramientas

HTML es un Lenguaje de Marcado de Hipertexto (HyperText Markup Language), es un estándar que se utiliza para la creación y el diseño de páginas web. Con HTML creamos la estructura de cada una de las secciones disponibles de la pagina web y la aplicación LADETH. Las secciones de la página web son: inicio, acerca de nosotros, servicios, contacto, politicas de privacidad, inicio de sesion y registro. En la aplicación tenemos diferentes secciones segun el tipo de usuario que inicie sesión dentro de la aplicación.

CSS significa Hojas de Estilo en Cascada (Cascading Style Sheets), es un lenguaje de diseño utilizado para controlar el aspecto y el formato del contenido HTML en páginas web. CSS permite separar la estructura y el contenido de un documento HTML de su presentación visual, lo que facilita la consistencia y el mantenimiento del diseño en un sitio web. Con el creamos los estilos para cada sección de la aplicación de LADETH haciendo que se adapte a cualquier dispositivo en el cual se accede.

JavaScript (JS) es un lenguaje de programación de alto nivel y de propósito general que se utiliza principalmente para el desarrollo de páginas web. Con agregamos animaciones y funcionalidad a la aplicación web LADETH, de igual manera gracias a JavaScript hacemos posible el uso de la aplicación sin conexión a internet haciendo la aplicación web progresiva.

Bootstrap es un marco de diseño (framework) front-end de código abierto creado por Twitter y mantenido actualmente por la comunidad de desarrolladores de software. Su objetivo principal es simplificar y agilizar el desarrollo web al proporcionar un conjunto de herramientas y estilos predefinidos que facilitan la creación de interfaces web modernas y responsivas.

PHP (Hypertext Preprocessor) es un lenguaje de programación del lado del servidor diseñado específicamente para el desarrollo web. Con PHP realizamos la conexión a la base de datos de la aplicación y nos permite usar el Modelo Vista Controlador para la separación de la lógica de la aplicación con el diseño.

MySQL es un sistema de gestión de bases de datos relacional (RDBMS, por sus siglas en inglés) que se utiliza para almacenar, organizar y gestionar datos de manera estructurada. Con MySQL creamos la base de datos y las tablas donde almacenamos la información de los usuarios y los servicios disponibles de la aplicación LADETH.

Git es un sistema de control de versiones distribuido ampliamente utilizado para el desarrollo de software. Git permite realizar un seguimiento de los cambios en el código fuente a lo largo del tiempo. Cada cambio o actualización de la aplicación se guardará en una versión diferente para evitar errores.

GitHub es una plataforma de desarrollo colaborativo basada en la web que utiliza Git como sistema de control de versiones. Proporciona un entorno en línea donde alojamos los archivos de la aplicación y colaboramos los desarrolladores en este proyecto de software.

GitKraken es una aplicación de interfaz gráfica de usuario (GUI) para Git, que proporciona una experiencia visual y amigable para gestionar repositorios Git. Con esta aplicación guardamos los cambios de cada versión de la aplicación LADETH.
