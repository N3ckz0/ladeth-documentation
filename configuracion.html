Configuración

Bootstrap

Para agregar estilos a la página web de LADETH primero debemos de instalar Bootstrap, descargamos los archivos de la página oficial los cuales vienen comprimidos en un archivo .zip, descomprimimos el archivo .zip y nos dejará 2 carpetas nombradas css y js. Cada carpeta contiene varios archivos pero solo conservaremos los archivos nombrados bootstrap.min.css que se encuentra en la carpeta css, bootstrap.bundle.min.js y bootstrap.min.js que se encuentran en la carpeta js. Agregamos primero el archivo bootstrap.min.css en una etiqueta link en el head de la pagina y los archivos bootstrap.bundle.min.js y bootstrap.min.js al final del body de la pagina.

Git

Para configurar Git debemos ingresar un nombre de usuario y un correo. Para esto abrimos la aplicación y configuramos primero el nombre con el comando git config --global user.name y despues escribimos el usuario entre comillas. Para configurar nuestro correo escribimos el comando git config --global user.email y despues escribimos el correo electrónico entre comillas. También podemos configurar el editor de Texto pero como usaremos GitKraken no es necesario.

GitKraken

Para configurar GitKraken solo debemos iniciar sesión con nuestra cuenta de GitHub y para abrir un repositorio debemos ingresar el link del repositorio en GitHub
